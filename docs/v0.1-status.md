# Mapeamento da v0.1 contra a especificação

## Entregue

| Requisito v0.1 | Implementação |
|---|---|
| Projeto `.rtsim` | `examples/basic/projeto.rtsim` + parser em `config.rs` |
| Leitura de `robot.json` | `config.rs` |
| Leitura de `track.json` | `config.rs` |
| Simulação fixed-step | `sim.rs`, loop por `physics_dt_us` em microssegundos |
| Modelo de pista simples | `track.rs`, `VectorTrack` por polilinha |
| Modelo de sensor simples | `sensor.rs`, `SensorArray16` simplificado |
| Modelo de motor DC simples | `motor.rs`, `DcMotorSimple` |
| Roda com Coulomb friction | `wheel.rs`, `CoulombFrictionWheel` |
| Modelo `NoDownforce` | `sim.rs`, `N = m*g` dividido entre esquerda/direita |
| Controlador PID built-in | `controller.rs`, `BuiltInPid` |
| Log CSV | `telemetry.rs` |
| Execução por terminal | `cli.rs`, comando `run` |
| Benchmark | `cli.rs`, comando `benchmark` |

## Comando-alvo

```bash
robotrace-sim run examples/basic/projeto.rtsim --headless --duration 10s
```

## Observação

O ambiente usado para gerar este pacote não possui toolchain Rust instalado, portanto a validação local foi feita por inspeção de estrutura e consistência dos arquivos. O próximo passo recomendado é rodar no ambiente de desenvolvimento:

```bash
cargo test
cargo run --release -- run examples/basic/projeto.rtsim --headless --duration 10s
cargo run --release -- benchmark examples/basic/projeto.rtsim --duration 10s --physics-dt-us 500
```
